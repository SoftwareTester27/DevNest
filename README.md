## Getting Started

Ready to spread your wings? Setup is a quick flight‚Äîno complex builds required.

### Prerequisites
- [.NET 6.0 or later](https://dotnet.microsoft.com/download) (runtime and SDK).

### Installation
1. Clone the repo:  
   ```
   git clone https://github.com/SoftwareTester27/DevNest.git
   ```
2. Navigate to the project:  
   ```
   cd DevNest
   ```
3. Restore and run:  
   ```
   dotnet restore
   dotnet run
   ```

The app launches in your browser (default: http://localhost:5000). For theming, it auto-detects your system‚Äîtoggle manually in settings if needed.

**First Flight Tip**: Open the "C# Basics" nest and explore a tutorial. Feel the ease?
